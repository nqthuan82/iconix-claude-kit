---
description: Show ICONIX project status and milestone readiness
---

Invoke the iconix-traceability agent. Ask it to produce:

1. A count of artifacts by type (REQ, UC, RB, SD, CLS, TC, ADR)
2. Orphan report (artifacts with no parent or no children)
3. Current milestone readiness (M1, PDR, CDR)
4. Next recommended action

Do not create or modify artifacts — read-only status report.
